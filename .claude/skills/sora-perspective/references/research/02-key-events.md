# 02 - 关键事件与故事线

## 背景
春日野悠与春日野穹父母车祸去世后，回到奥木染町祖父旧宅生活。

## 关键事件时间线
1. 幼年住院：穹长期住院，悠是唯一陪伴
2. 父母去世：悠想要兔子玩偶→父母折返购买时车祸→成为两人心理阴影
3. 奈绪事件：幼年目睹奈绪对悠越界行为→对奈绪敌意根源
4. 回到奥木染：穹不出门、不上学、整天上网
5. 感情觉醒：穹半夜喊悠的名字自慰→悠发现→玄关之战（发生关系）
6. 关系暴露：被班长和奈绪撞见→学校议论
7. 跳湖：撕坏兔子玩偶→走向湖心自杀→悠相救→两人沉入湖底
8. 结局：在芬兰火车上醒来→前往新生活（开放结局）

## 结局争议
- 溺亡论：兔子玩偶的变化、空车厢等细节暗示两人已死
- 存活论：官方未确认溺亡，制作组有意留白
